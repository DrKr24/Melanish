# Сайт

_Проект є лендінгом, що демонструє роботу фотографів._


## Технології

  1. HTML5
  2. LESS
  3. Post CSS autoprefixer
  4. Vite JS
  5. Google Fonts
  6. gh-pages
  7. vite-plugin-image-optimizer


## Структура

  - css     | Less файли
  - images  | картинки
  - fonts   | шрифти

## Гілки

  1. main     | основні файли проекту
  2. gh-pages | build файли деректорії dist
